TECHNOLOGIES UTILISÉES
======================

- HTML5 / CSS3 / JavaScript Vanilla
- Chrome Extension APIs:
  * chrome.storage (stockage local)
  * chrome.declarativeNetRequest (blocage)
  * chrome.notifications (notifications)
  * chrome.alarms (rappels quotidiens)
  * chrome.commands (raccourcis clavier)
  * chrome.runtime (messages, options)

INSTALLATION
============

1. Crée les 3 icônes PNG (icon16.png, icon48.png, icon128.png)
2. Va à chrome://extensions/
3. Active "Mode développeur"
4. Clique "Charger l'extension non empaquetée"
5. Sélectionne le dossier extension-nnn
6. Profite de l'extension !
