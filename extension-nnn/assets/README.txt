PLACEHOLDER ICONS

Pour utiliser cette extension, vous devez créer les icônes suivantes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Design suggéré:
- Fond bleu foncé (#1a237e)
- Cercle vert (#4caf50) au centre
- Lettre "N" blanche au centre

Vous pouvez utiliser generate-icons.html dans le dossier parent ou tout outil de design.
