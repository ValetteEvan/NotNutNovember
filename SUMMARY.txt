║ 🚀 INSTALLATION (3 ÉTAPES)                                                 ║
╚════════════════════════════════════════════════════════════════════════════╝

  ÉTAPE 1 : Créer les icônes (1 minute)
  ──────────────────────────────────────
    → Ouvrir extension-nnn/generate-icons.html
    → Sauvegarder les 3 canvas dans assets/
      • icon16.png
      • icon48.png  
      • icon128.png

  ÉTAPE 2 : Charger l'extension (1 minute)
  ─────────────────────────────────────────
    → Chrome : chrome://extensions/
    → Activer "Mode développeur"
    → "Charger l'extension non empaquetée"
    → Sélectionner le dossier extension-nnn

  ÉTAPE 3 : Lancer le challenge (30 secondes)
  ────────────────────────────────────────────
    → Cliquer sur l'icône de l'extension
    → Cliquer "Valider aujourd'hui"
    → C'est parti ! 🚀
